# s34841.github.io
ยินดีต้อนรับเข้าสู่แฟ้มสะสมผลงานของ นางสาวนภัสพร สมกระบวน
ปัจจุบันกำลังศึกษาอยู่ระดับชั้นมัธยมศึกษาปีที่6 แผนการเรียน Gifted Program โรงเรียนบ้านบึง"อุตสาหกรรมนุเคราะห์" ดิฉันชอบเรียนรู้สิ่งใหม่ๆ และเข้าร่วมกิจกรรมที่ช่วยเสริมสร้างความรู้และประสบการณ์ แฟ้มสะสมผลงานเล่มนี้จัดทำขึ้นเพื่อรวบรวมผลงาน ความสามารถ และความภาคภูมิใจที่ได้สั่งสมมา หวังเป็นอย่างยิ่งว่าแฟ้มสะสมผลงานนี้จะแสดงให้เห็นถึงความตั้งใจและศักยภาพของดิฉันได้เป็นอย่างดีค่ะ
